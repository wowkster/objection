# Objection

A simple object storage server written in Rust using Tokio and SQLite. The goal is to be 100% compatible with AWS S3's APIs while making it easy to adminstrate and interact with.

## Requirements

- AWS S3 API Compatibility
- Per Bucket + Per Object Access Control
- CORS
- TLS
- HIGH Performance
- Cache Control
- Bucket Data Deduplication
- Content Types
- Optional Access Logging
- Optional Object Expiration
- Object Tagging
- Easy Backups
- IP Rate Limiting
